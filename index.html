Ресурспак КСЭПСП - это набор текстур и ресурсов для Minecraft: Java Edition 1.21.5+. Он предназначен для изменения текстур и моделей предметов и сущностей в игре путём их переименования. The "KSEPSP" Resource Pack - is a pack of textures and resources for Minecraft: Java Edition 1.21.5+. It is designed to modify the textures and models of items and entities in the game through renaming.

Набор использует ванильные возможности для создания кастомных текстур предметов. В скором будущем планируется поддержка переименования брони с использованием модификаций ETF и EMF. The pack uses vanilla features to create custom item textures. In the near future, support is planned for armor renaming using the modifications ETF and EMF.

ИнструкцииInstructions

Для использования большей части набора ресурсов НЕ требуется установка модов. Для возможности переименования сущностей требуется установить модификации ETF и EMF. For most of the resource pack to work, installing mods is NOT required. However, to enable entity renaming, you must install the modifications ETF and EMF.

Для скачивания представлены две версии ресурспака: 9.5 для версий Майнкрафта после 1.21.5 и 9.0 для версий ниже. В 9.0 нет некоторых вещей, присутствующих на сайте. Two versions of the resource pack are available for download: 9.5 for versions 1.21.5 and above and 9.0 for versions vefore 1.21.5. 9.0 does not have some things presented on the website.

Для установки набора требуется скопировать загруженный архив с текстурами в папку resourcepacks в коренной папке Minecraft, и далее включить его в настройках игры в соответствующем меню.To install the resource pack you need to copy the downloaded archive to the "resourcepacks" folder in Minecraft root folder and turn it on in the game settings.

Для получения изменённых предметов, требуется переименовать их на наковальне. Для получения средств передвижения, требуется переименовать биркой определённую сущность.To get custom items, you need to rename them on the anvil in game. To get vehicles, you need to rename a certain entity with a name tag.

Если у вас возникли проблемы с набором ресурсов вы можете попытаться найти решение в официальной группе в ВК в обсуждении "Баг-трекер", в постах на стене сообщества или в официальном Телеграмм канале. Если вы не нашли там решения своей проблемы то можете обратится напрямую к нам через личные сообщения. If you encounter issues with the resource pack, you can try finding a solution in the official VK group under the "Bug Tracker" discussion, in community wall posts, or in the official Telegram channel. In case you don’t find a solution there, feel free to contact us in DM.
