Ресурспак КСЭПСП - это набор текстур и ресурсов для Minecraft: Java Edition 1.17 - 1.20+. Он предназначен для изменения текстур и моделей предметов и сущностей в игре путём их переименования.The "KSEPSP" Resource Pack - is a pack of textures and resources for Minecraft: Java Edition 1.17 - 1.20+. It both changes some original Minecraft textures and replaces many of them when renaming an item or entity.

Набор использует функцию кастомных текстур предметов (CIT), модификации Optifine.The pack uses the function of custom items (CIT), Optifine modification.

Урезанная и полная версия набора отличаются наличием в последней изменением некоторых текстур переименованных предметов, а так же наличием возможности переименования сущностей.
Учитывайте это при скачивании ресурспака.The cut and full versions of the pack differ by having some changed textures some of the renamed items and abillity of renaming entities.
Consider that before downloading the resource pack.

ИнструкцииInstructions

Для использования набора ресурсов необходимо установить модификацию Optifine или заменяющие её функционал модификации CIT Resewn, Entity Model Features и Entity Texture Features. Без них большая часть набора ресуров работать не будет.For using the resource pack you need to download the Optifine modification or modifications replacing its functions CIT Resewn, Entity Model Features и Entity Texture Features. Without them major part of resource pack content will not work.

Для установки набора требуется скопировать загруженный архив с текстурами в папку resourcepacks в коренной папке Minecraft, и далее включить его в настройках игры в соответствующем меню. Для возможности переиманования сущностей следует при установленном Optifine включить в настройках графики в разделе 'Качество" пункт "Кастомные мобы" или установить соответсвующие модификации, если ваша сборка не включает в себя Optifine.To install the resource pack you need to copy the downloaded archive to the "resourcepacks" folder in Minecraft root folder and turn it on in the game settings.

Для получения изменённых предметов, требуется переименовать их на наковальне. Для получения средств передвижения, требуется переименовать биркой определённую сущность.To get custom items, you need to rename them on the anvil in game. To get vehicles, you need to rename a certain entity with a name tag.

Если у вас возникли проблемы с набором ресурсов вы можете попытаться найти решение в официальной группе в ВК в обсуждении "Баг-трекер" или в постах на стене сообщества. Если вы не нашли там решения своей проблемы то можете обратится напрямую к нам через личные сообщения сообщества.If you have any problems with the resource pack, you can try to find a solution in the official VK group(rus) in the Bug Tracker discussion or in posts on the wall. If you have not found a solution to your problem there, then you can contact us directly through personal messages.
